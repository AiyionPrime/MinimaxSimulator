<a name="compiling-source"></a> Compiling from Source
------------------------------------------------------
#### Prerequirements (newer version should also work)
* JDK 8u40
* Apache Maven 3.0.5

#### Compilation
The recommended way to compile the simulator from source code is:
```bash
mvn clean compile assembly:single
```
